

**Basic Matrix Coding Questions:**

1. **Transpose a Matrix:** Write a function to find the transpose of a given matrix.
2. **Addition of Matrices:** Write a function to add two matrices of the same dimensions.
3. **Multiplication of Matrices:** Write a function to multiply two matrices.
4. **Scalar Multiplication:** Write a function to multiply a matrix by a scalar.
5. **Check for Symmetry:** Write a function to check if a matrix is symmetric.

**Intermediate Matrix Coding Questions:**

6. **Spiral Order Matrix Traversal:** Write a function to traverse a matrix in a spiral order.
7. **Rotate a Matrix:** Write a function to rotate a matrix by 90 degrees (clockwise or counterclockwise).
8. **Search in a Sorted Matrix:** Given a sorted matrix, write a function to search for a target value.
9. **Set Matrix Zeroes:** Given a matrix, if an element is 0, set its entire row and column to 0.
10. **Find the Diagonal Elements:** Write a function to find the diagonal elements of a matrix.

**Advanced Matrix Coding Questions:**

11. **Matrix Chain Multiplication:** Given a sequence of matrices, find the most efficient way to multiply them.
12. **Kth Element of a Matrix:** Given an n x n matrix, where each of the rows and columns are sorted, find the kth smallest element.
13. **Rotate Image (NxN matrix):** Rotate an n x n 2D matrix (image) by 90 degrees.
14. **Search a 2D Matrix II:** Write an efficient algorithm to search for a target value in an m x n matrix.
15. **Largest Submatrix with all 1s:** Given a binary matrix, find the largest square submatrix with all 1s.

**Challenging Matrix Coding Questions:**

16. **Maximum Size Rectangle of 1s:** Given a binary matrix, find the largest rectangle containing only 1s.
17. **Sparse Matrix Multiplication:** Given two sparse matrices, return the result of multiplication.
18. **Finding All Connected Components:** Given a 2D matrix representing a graph, find all connected components.
19. **Floyd-Warshall Algorithm:** Implement the Floyd-Warshall algorithm for all pairs shortest paths in a weighted graph represented by a matrix.
20. **Convex Hull in a Matrix:** Given a binary matrix, find the maximum area of a rectangle formed by 1s.
